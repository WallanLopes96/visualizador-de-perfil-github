# visualizador-de-perfil-github

Endpoints que vmaos usar

Perfil: Get /users/:username

Repositorios: Get /users/:username/repos (usar query per_page=10 e sort=updated / sort=pushed conforme objetivo)

# Funcionaidades do projeto:
# 1. Buscar Usuários do GitHUb pelo nome de usuário
# 2. Exibir informações de perfil (avatar, nome, bio, seguidores e seguindo)
# 3. Exibir os 10 ultimos repositórios com statísticas (stars, forks, watchers, linguagem)
# 4. Tratar estados de loading, erro e usuários não encontrados.
# 5. Interface responsiva e animações suaves
